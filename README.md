#### 👋 Hello and welcome to my GitHub profile

I'm a Full-Stack JavaScript developer specialized in **AI integration and automation** for real production systems. I work end-to-end — React, Next.js, Node.js, and Python on the application side; OpenAI, xAI, Anthropic, Perplexity, and agentic pipelines on the AI side.

Most teams know they need AI but not *where* it actually creates value. That's the part I focus on: I come into a system or workflow, find where AI and automation create real leverage (and where they shouldn't be used), then build the solution end-to-end. Before tech, I spent 17 years leading operations — which is why I think in systems and care about solutions that survive real-world use.
